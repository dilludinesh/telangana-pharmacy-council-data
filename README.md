# TGPC Rx

Automated system for Rx record management.

## Overview

- Daily Rx data extraction
- Validation and deduplication
- Cloud sync
- JSON backup

## Data Points

- Serial
- Registration ID
- Name
- Parent Name
- Category

## Architecture

- **Core**: Python logic
- **Auto**: GitHub Actions
- **Cloud**: Supabase
- **Web**: Search interface

## Usage

Automated execution.
